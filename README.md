# Python Fundamentals Training - Part 2

This repository contains Python training materials and practice problems focused on **Loops** and **Functions**, which are key building blocks for writing efficient and reusable code.

---

## Contents

### 1. Loops & Loop Control Statements
- Understanding **definite loops** (`for` loops)
- Understanding **indefinite loops** (`while` loops)
- Using **break** and **continue** to control loop flow

### 2. Python Functions
- Defining functions with `def`
- Using **return values**
- Function parameters and default arguments
- Handling variable-length argument lists with `*args` and `**kwargs`
- Understanding **local** vs **global** variable scope

### 3. Practice Problems & Solutions

This folder contains Python scripts implementing common programming problems to practice loops and functions:

| Problem              | Description                                       | Filename                  |
|----------------------|-------------------------------------------------|---------------------------|
| Palindrome Checker   | Check if a string reads the same forwards/backwards | `palindrome_checker.py`    |
| FizzBuzz             | Print numbers with Fizz/Buzz rules               | `fizzbuzz.py`              |
| Sum of Digits        | Sum the digits of an integer                      | `sum_of_digits.py`         |
| Multiplication Table | Print multiplication table for a number          | `multiplication_table.py`  |
| Factorial            | Calculate factorial of a non-negative integer    | `factorial.py`             |
| Fibonacci Sequence   | Print first n Fibonacci numbers                   | `fibonacci.py`             |
| Prime Number Check   | Check if a number is prime                         | `prime_number.py`          |
| Word Counter         | Count number of words in a string                 | `word_counter.py`          |
| Vowel Remover        | Remove all vowels from a string                    | `vowel_remover.py`         |
